## 前言

時不時會看一些已經公開的資安漏洞，這次剛好看到新的名詞 VNC，就上網查了一下

## 簡介

|      | VNC                       |
| ---- | ------------------------- |
| Port | 5900 ~ 59xx               |
| 全名 | Virtual Network Computing |
