從 2025/03/09 到 2025/07/17，經歷了 4 個月的時間，終於完成 HTTP 30 篇文章的挑戰。中間我其實有跑去研究資安，但後來又覺得要先把基礎知識學會，在資安領域才能夠更深入，於是大概在 6 月中開始，我又回來開始瘋狂寫文章，想趕快把階段性的任務（30 篇文章）完成，之後就可以沒有進度壓力的繼續研讀 HTTP 以及其他主題了。

現在如果回頭考我文章內談到的各種小細節，我可能還是會需要時間喚起記憶，但至少不是完全不知道，也許某天真的要用到這些知識的時候，我可以回頭翻這些文章，並且順便更新內容。隨著我們學習的知識越來越多，當未來真的遇到技術決策的時候，我們腦袋內會有更多的選擇，而不是只侷限在自己知道的那些 Solution。

經過這個 30 篇文章的挑戰，我才發現 Web 的世界很廣，光是 HTTP 就有很多我沒接觸過的新知識，學了越多，才發現自己很渺小，不知道的事情太多了，因為知識跟知識之間是可以一直延伸的。我剛開始定義 30 篇文章的主題，覺得好難湊滿，但寫到後來，學了 CORS 又發現還有 CORP。當我打開一扇門，又發現有更多扇門時，那種感覺真的是難以言喻，會慢慢意識到自己的見識不夠廣，並且想要繼續努力投資自己的腦袋。

HTTP 的路還沒結束，未來我還是會不定期發布新的主題。截止 2025/07/17，我總共定義了 42 篇的 HTTP 主題，代表我還有 12 篇還沒完成呢！當然，舊的文章也會不定期更新，如果你有發現過時的文件或敘述，也歡迎讓我知道呦！最後，感謝認真看完 30 篇 HTTP 文章的你，希望看完以後有讓你學到新東西！
